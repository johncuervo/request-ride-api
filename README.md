# Sinatra REST API

Install all the gems
- bundle install

Create DB
Once you've created and configured the config/database.yml file, you can create
the database.

bundle exec rake db:create
bundle exec rake db:migrate
bundle exec rake db:seed

Run Aplication
-bundle exec rerun app.rb
